COVID-19

Data

Visualization

This was a project I created to help myself - and others - visualize how the on-going pandemic is effecting Canada and the rest of the world.

The project was built using Python Flask, Dash and Plotly. All of the COVID-19 data is pulled from 2 seperate APIs. The first API is to get specific data in relation to the provinces in Canada. The second API pulls COVID-19 data for the rest of the world. Flask-Caching was used to help prevent the amount of times the program reaches out to pull COVID-19 data from the APIs, the data is cached for 5 minutes. Flask-Caching is also used to cache the GeoJSON data (being used to render the Canadian map) to prevent the program from constantly reading from the GeoJSON file, this is cached for 30 minutes.
